A curated dashboard of open-source OSINT and threat-intelligence tools.

Live site: https://recuon.github.io/threat-intel-osint-dashboard/

All tool metadata (stars, language, license, last-updated) was verified against the GitHub API on 2026-07-06. The dashboard lists 34 open-source tools across 8 categories.

## Categories

- OSINT Recon & Footprinting (4)
- Username/People & Email Enumeration (5)
- Network & Attack-Surface Mapping (5)
- Threat Intelligence Platforms (4)
- Malware Analysis & Sandboxing (4)
- Detection Engineering & Hunting (4)
- SOC/IR & Case Management (4)
- Feeds/Enrichment & Orchestration (4)

## How the data was verified

Every tool's stars, language, license, and last-updated timestamp was fetched from the GitHub REST API on 2026-07-06. The page is a single self-contained static HTML file with zero external runtime requests.
