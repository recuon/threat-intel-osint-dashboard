A curated dashboard of open-source OSINT and threat-intelligence tools.

Live site: https://recuon.github.io/threat-intel-osint-dashboard/

All tool metadata (stars, language, license, last-updated) was verified against the GitHub API on 2026-07-06. The dashboard lists 52 open-source tools across 8 categories.

## Categories

- OSINT Recon & Footprinting (6)
- Username/People & Email Enumeration (7)
- Network & Attack-Surface Mapping (8)
- Threat Intelligence Platforms (6)
- Malware Analysis & Sandboxing (6)
- Detection Engineering & Hunting (6)
- SOC/IR & Case Management (6)
- Feeds/Enrichment & Orchestration (7)

## How the data was verified

Every tool's stars, language, license, and last-updated timestamp was fetched from the GitHub REST API on 2026-07-06. The page is a single self-contained static HTML file with zero external runtime requests.
